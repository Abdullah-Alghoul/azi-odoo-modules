Module name

This module was written to extend the functionality of MRP Time Bucket to support BOM effectivity dates and allow you to schedule against BOMs planned in the future.

Usage

Set BOM validity dates in the future and verify that the correct BOM is chosen against the Scheduled Date on a Manufacturing Order.

For further information, please visit:

  • https://github.com/asphaltzipper/azi-odoo-modules/tree/master/mrp_bom_effectivity
  • https://www.odoo.com/forum/help-1
