MRP Material Analysis

This module was written to extend the functionality of Material Requirements Planning to find incomplete transactions for a given product and master schedule.

Installation

To install this module, you need to:

  • Click "Install"

Configuration

No configuration required.

Usage

Run the wizard located at Stock > Planning > Material Analysis.

For further information, please visit:

  • https://github.com/asphaltzipper/azi-odoo-modules/tree/master/mrp_material_analysis
